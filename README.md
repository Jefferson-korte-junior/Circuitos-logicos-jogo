# Logic Circuits Game - Projeto Integrador I

Um jogo educacional 2D sobre **circuitos lógicos**, desenvolvido como parte do Projeto Integrador I na UTFPR - Campus Santa Helena.  
Criado com a engine **Godot** e linguagem **GDScript**, o objetivo do jogo é ensinar os conceitos básicos de lógica digital de forma lúdica e interativa.

---

## Demonstração

>  O jogo foi exportado como APK e testado em dispositivos Android.

---

##  Tecnologias Utilizadas

- **Godot Engine**  
- **GDScript**  
- **Git e GitHub**  
- **Metodologias ágeis (Scrum/Kanban)**

---

## Funcionalidades

- Apresentação de portas lógicas (AND, OR, NOT, etc)
- Resolução de desafios com lógica digital
- Interface 2D com elementos interativos
- Sistema de feedback e pontuação

---

## Estrutura do Projeto

/assets -> sprites, áudio e fontes
